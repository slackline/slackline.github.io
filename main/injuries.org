*** Personal Activities

Following the advice in  [[https://www.youtube.com/watch?v=fuFyPHI4he4][How to Fix Medial Elbow Pain (Golfer's / Climber's Elbow) in 26 Minutes!]] the following
exercises should be undertaken. See also earlier [[https://www.youtube.com/watch?v=L0ECHTgc4TM][Ulnar Nerve Pain]] video.

| Action                           | Volume                              | Frequency                                | Discomfort Level | Notes                                                                                                                                                                                                                                                                                                                                                               |
|----------------------------------+-------------------------------------+------------------------------------------+------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Soft Tissue Massage              | 3-5 minutes                         | daily or multiple daily                  | 2/10             | Stop if it hurts                                                                                                                                                                                                                                                                                                                                                    |
|----------------------------------+-------------------------------------+------------------------------------------+------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Tissue Stetches...               |                                     |                                          |                  |                                                                                                                                                                                                                                                                                                                                                                     |
| Flexor stretch + nerve glide     | 3-5reps                             | multiple times a day as tolerated        | 2/10             | With arm bent at 90, pull fingers back towards bicep and slide arm forwards and backwards. Attenuate extremes to accommodate pain.                                                                                                                                                                                                                                  |
| Pec Stretch 90 + 120             | 2-3reps at both angles on both arms | multiple times a day as tolerated        | 2/10             | In a doorway, elbow level with shoulder and arm bent (90) put arm on frame, step forward to move body and open pecs, look down on opposite side, push forward with back foot. Repeat at 120 degrees                                                                                                                                                                 |
|----------------------------------+-------------------------------------+------------------------------------------+------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Nerve Mobilisation...            | 1-2 sets of 3-4reps per exercise    | 2-3 times a week to start then increases |                  |                                                                                                                                                                                                                                                                                                                                                                     |
| Low Median nerve floss + tension |                                     |                                          | 2/10             | Start with hands curled and elbows bent in front, extend arms and lower them then push back behind the body, raising palms to horizontal (facing the floor), straighten fingers and point back. Then look into armpit. Return and repeat (both sides is one rep). Keep shoulders down and just intensity by bending elbows or not extending/pushing fingers as far. |
| High Median floss + tension      |                                     |                                          | 2/10             | Arms horizontal level with chest and bent at the elbow, hands relaxed. Straighten arms out sideways and point fingers up. Rotate and look at alternate sides to tension the opposite side and floss (loosen) the side you're turning into. Keep shoulders down                                                                                                      |
| Ulnar nerve floss + tension      |                                     |                                          | 2/10             | Palms together, shoulders down, lift hands to under chin and lower. When hands are low alternate looking to opposite sides. Move both hands to one side then the other, combine with looking in opposite direction.                                                                                                                                                 |
|----------------------------------+-------------------------------------+------------------------------------------+------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Tissue Loading...                | 8-10 reps per exercise              | 2-3 times a week                         |                  | Have two repetitions in reserve                                                                                                                                                                                                                                                                                                                                     |
| Forearm pronation                |                                     |                                          |                  | Weight in hand and rotate outwards and back to vertical. If arm/wrist is twisting coming back too much weight. Aim for full range of motion but not a problem if not quite reaching it.                                                                                                                                                                             |
| Wrist flexion                    |                                     |                                          |                  | Supported (under elbow) bicep curls, avoid twisting on curling.                                                                                                                                                                                                                                                                                                     |
| Bicep curls                      |                                     |                                          |                  | Start with arms natural (thumbs forward) holding weights, bring up and twist hands so fingers are facing shoulder.                                                                                                                                                                                                                                                       |
| Finger curls                     |                                     |                                          |                  | Weights at side with wrist curled, uncurl, but don't drop them!                                                                                                                                                                                                                                                                                                     |
