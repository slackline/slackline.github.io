#+TITLE: DeltaChat
#+FILETAGS: :privacy:email:encryption:

[[https://delta.chat/en/][DeltaChat]] is a "/decentralized secure messenger/" that uses email and encryption (Rust PGP to be specific) to send
messages in near real-time between users. It uses its own Chatmail servers to relay messages.

* Install

** Android

** Gentoo

** Arch

#+begin_example shell
pacman -Syu deltachat
#+end_example
