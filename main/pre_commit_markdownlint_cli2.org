* Ignoring Files

Ignore directories of groups of files using globs by adding the ~ignores~ section to ~.markdownlint-cli2.yaml~

#+begin_src yaml
  ignores:
    - "*.svg"
    - "_site/*"
    - ".quarto/*"
#+end_src
