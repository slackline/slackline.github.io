#+TITLE: Oyo
#+FILETAGS: :linux:alternative:diff:

[[https://github.com/ahkohd/oyo][~oyo~]] is an alternative diff tool (in the same vein as ~difftastic~). It supports stepping through changes as well as
"split" view and [[id:3c905838-8de4-4bb6-9171-98c1332456be][Git]] integration

* Install

** Arch

#+begin_example
git clone https://aur.archlinux.org/oyo.git
cd oyo
makepkg -sri
#+end_example

** Gentoo

Not yet in overlays.
