#+TITLE: Git Attributes
#+FILETAGS: :git:configuration:config:attributes:

The ~.gitattributes~ file is a special file that helps define how a repository behaves across operating system. It is
useful for setting End Of Line (~eol~) types.

It works by specifying a pathname (or glob) and then setting the desired attributes.

#+begin_example
.git_archival.txt  export-subst
*.py text eol=lf
*.yaml text eol=lf
#+end_example
