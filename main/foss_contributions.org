* Work

** [[https://github.com/AFM-SPM/TopoStats][TopoStats]]

My first major project when I started work as an RSE was overhauling a horrendous Python 2 based Docker containerised
workflow for processing Atomic Force Microscopy images.

Contributions are too numerous to list but I've had a major role in reshaping the code base ([[https://github.com/AFM-SPM/TopoStats/commits?author=ns-rse][397 commits; 2,064,265
++30,508 --]]) and helped the team adopt good software engineering practices. Massive help from my friend [[https://github.com/SylviaWhittle][SylviaWhittle]]
who took on much of the  hard-core coding.
