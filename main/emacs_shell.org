#+TITLE: Emacs Shell
#+FILETAGS: :emacs:shell:

There are lots of terminal/shell emulators available such as [[id:4668a662-27ab-4015-aaa0-6e356b6bc165][Emacs vterm]].

* Run Shell commands

You can run shell commands with ~M-x shell-command~ (bound to ~M-!~ by default). You can also insert the results of the
shell command at the current point using ~C-u M-!~, magic!
