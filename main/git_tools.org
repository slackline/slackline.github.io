#+TITLE: Git Tools
#+FILETAGS: :git:extensions:tools:

By far the most powerful tool is [[id:220d7ba9-d30e-4149-a25b-03796e098b0d][Magit]] but outside of that there are many additional and useful options.

* Git Time Machine

[[https://dinakars777.github.io/git-time-machine/][Git Time Machine]] is a "/Visual Git Reflog Browser/".

* GitHub Profiler

Grab some summary information about a user using the [[https://github.com/ehmatthes/gh-profiler][gh-profiler]] tool.

#+begin_example
uvx gh-profiler ns-rse
#+end_example

* Git Lint

[[https://alchemists.io/projects/git-lint][git-lint]] lints your commit history to give a consistent style.

* Managing Multiple Repositories

** My Repos

 [[https://myrepos.branchable.com/][myrepos]] is a tool for managing multiple repositories.

** Gitopolis

[[https://github.com/timabell/gitopolis][gitopolis]] is another tool, written in [[id:3469c33e-7c61-46c7-b01e-655695f3b93c][Rust]], for managing multiple repositories.
